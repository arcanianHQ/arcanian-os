# How to Create Custom Fields in Asana

**Source:** https://help.asana.com/s/article/custom-fields

**Note:** Custom fields are an Asana Premium feature.

---

## Method 1: Via Customize Menu (Recommended)

1. Open the project where you want to add the field
2. Click **"Customize"** (top right of project)
3. Under "Add" section, click **"Fields"**
4. Enter field name (e.g., "Service Type")
5. Choose field type:
   - **Dropdown** - Select from predefined options (up to 500 choices)
   - **Checkbox** - Simple yes/no
   - **Text** - Open-ended responses
   - **Number** - Quantities, budgets, metrics
6. Add options (for dropdown)
7. Set colors for each option (optional)
8. **Important:** Check "Add to organization's field library" to make it reusable
9. Click **"Create field"**

---

## Method 2: Via List View

1. Open project in **List view**
2. Click the **+** button next to the row of field headers
3. Follow same configuration steps as above

---

## Adding Existing Field to Another Project

1. Open the target project
2. Click **"Customize"** (top right)
3. Click **"Add Field"**
4. Search for existing field name
5. Select to add it to the project
