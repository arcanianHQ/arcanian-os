# Asana Onboarding

## Overview

Asana is used for project and task management at Arcanian.

- **Workspace:** arcanian.ai
- **Lead Account:** laszlo.fazakas@arcanian.ai

## Onboarding Steps

### 1. Send Invitation

1. Log in to Asana as admin
2. Go to Admin Console → Members
3. Click "Invite members"
4. Enter new member's @arcanian.ai email
5. Select appropriate team(s):
   - **Arcanian Consulting** - for client project access
   - Other teams as needed

### 2. Team Assignment

| Role | Teams | Access Level |
|------|-------|--------------|
| Co-founder | All teams | Admin |
| Co-worker | Arcanian Consulting | Member |
| Client | Specific project only | Guest |

### 3. Project Access

- Assign to relevant projects based on role
- Follow naming convention (see R001 in RULES.md)

### 4. New Member Tasks

New member should:

1. [ ] Accept email invitation
2. [ ] Complete Asana profile
3. [ ] Download Asana mobile app (optional)
4. [ ] Review assigned projects

## Access Removal

1. Go to Admin Console → Members
2. Find member and click "Remove from workspace"
3. Reassign any owned tasks/projects
