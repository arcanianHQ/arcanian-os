# Akiflow Onboarding

## Overview

Akiflow is used for daily to-do management and time boxing.

- **Lead Account:** laszlo.fazakas@arcanian.ai
- **Primary User:** László Fazakas
- **Auth Method:** Email

## Onboarding Steps

### 1. Create Account

1. Go to https://akiflow.com
2. Sign up with @arcanian.ai email
3. Choose subscription plan

### 2. Initial Setup

1. Connect calendar (Google Calendar)
2. Set up daily planning preferences
3. Configure time blocking defaults
4. Set working hours

### 3. Integrations

Akiflow integrates with:
- Google Calendar
- Asana
- Slack
- Gmail
- Notion

To connect:
1. Go to Settings → Integrations
2. Select service to connect
3. Authorize access

### 4. New Member Tasks

1. [ ] Create Akiflow account
2. [ ] Connect Google Calendar
3. [ ] Connect Asana (for task sync)
4. [ ] Set up daily planning routine
5. [ ] Configure keyboard shortcuts

## Daily Workflow

1. Morning: Plan the day using time blocks
2. During day: Capture tasks from email/Slack
3. End of day: Review completed tasks

## Access Removal

1. Remove integrations first
2. Cancel subscription
3. Delete account in settings
