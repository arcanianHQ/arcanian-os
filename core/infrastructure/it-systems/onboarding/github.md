# GitHub Onboarding

## Overview

GitHub is used for code repositories and version control.

- **Organization:** ArcanianAi
- **Lead Account:** info@arcanian.ai

## Onboarding Steps

### 1. Prerequisites

New member needs:
- GitHub account (personal or create new)
- Two-factor authentication enabled

### 2. Send Organization Invitation

1. Go to github.com/orgs/ArcanianAi/people
2. Click "Invite member"
3. Enter GitHub username or email
4. Select role:
   - **Owner** - Full admin access (co-founders only)
   - **Member** - Standard access

### 3. Repository Access

After joining organization:

1. Go to repository settings
2. Manage access → Add people
3. Select permission level:
   - **Read** - View only
   - **Write** - Push commits
   - **Admin** - Full repository control

### 4. New Member Tasks

New member should:

1. [ ] Accept organization invitation
2. [ ] Set up SSH keys or use HTTPS
3. [ ] Clone required repositories
4. [ ] Configure git identity:
   ```bash
   git config --global user.name "Name"
   git config --global user.email "name@arcanian.ai"
   ```

## Access Removal

1. Go to organization → People
2. Find member → Remove from organization
3. Review and transfer any owned repositories
