# Forklift Onboarding

## Overview

Forklift is a dual-pane file manager for macOS with remote server support.

- **Platform:** macOS only
- **License:** Per-user license

## Onboarding Steps

### 1. Install Forklift

Option A - Mac App Store:
1. Open Mac App Store
2. Search "Forklift - File Manager"
3. Purchase and install

Option B - Direct Download:
1. Go to https://binarynights.com
2. Download Forklift
3. Purchase license

### 2. Initial Configuration

1. Open Forklift
2. Set up dual-pane layout (View → Dual Pane)
3. Configure favorites for common folders

### 3. Add Remote Connections (Optional)

Forklift supports:
- SFTP
- FTP
- Amazon S3
- Google Drive
- WebDAV

To add connection:
1. Go to Favorites → New Favorite
2. Select protocol
3. Enter server details and credentials
4. Save to favorites

### 4. New Member Tasks

1. [ ] Install Forklift from App Store or direct
2. [ ] Activate license
3. [ ] Configure dual-pane layout
4. [ ] Add required remote connections
5. [ ] Set up sync favorites (if applicable)

## License Management

- Licenses are per-user
- Contact admin for license purchase
- Transfer license via BinaryNights account if user leaves
