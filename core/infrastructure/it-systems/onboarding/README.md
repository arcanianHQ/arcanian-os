# Onboarding Documentation

This folder contains step-by-step onboarding guides for each system used at Arcanian.

## Purpose

When onboarding a new team member or client, use these guides to ensure proper setup and access to all required systems.

## Onboarding Guides

| System | File | Description |
|--------|------|-------------|
| Asana | [asana.md](asana.md) | Project management setup |
| Akiflow | [akiflow.md](akiflow.md) | Daily to-do & time boxing |
| GitHub | [github.md](github.md) | Code repository access |
| Linear | [linear.md](linear.md) | Issue tracking & project management |
| Zoom | [zoom.md](zoom.md) | Teleconferencing setup |
| Tailscale | [tailscale.md](tailscale.md) | VPN mesh network setup |
| Ubiquiti/UniFi | [ubiquiti.md](ubiquiti.md) | Network infrastructure access |
| SSH | [ssh.md](ssh.md) | Remote terminal access |
| Screen Sharing | [screen-sharing.md](screen-sharing.md) | Remote GUI access |
| Forklift | [forklift.md](forklift.md) | File manager setup |

## Onboarding Checklist

For new team members, complete these in order:

1. [ ] Google Workspace account created
2. [ ] Asana workspace invitation sent
3. [ ] GitHub organization invitation sent
4. [ ] Tailscale invitation sent (if remote access needed)
5. [ ] Required systems configured per individual guides
