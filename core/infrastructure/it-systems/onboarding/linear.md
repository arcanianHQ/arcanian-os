# Linear Onboarding

## Overview

Linear is used for issue tracking and project management.

- **Lead Account:** laszlo.fazakas@arcanian.ai
- **Workspace:** https://linear.app/arcanian
- **Auth Method:** SSO/Email

## Workspace Structure

| Team | Code | URL |
|------|------|-----|
| GRA | GRA | https://linear.app/arcanian/team/GRA/active |

## Onboarding Steps

### 1. Send Invitation

1. Go to Settings → Members
2. Click "Invite members"
3. Enter @arcanian.ai email
4. Select role and team(s)

### 2. Role Assignment

| Role | Permissions |
|------|-------------|
| Admin | Full workspace control |
| Member | Create/edit issues, access all teams |
| Guest | Limited access to specific projects |

### 3. Team Assignment

Assign new member to relevant teams:
- GRA - Main team

### 4. New Member Tasks

1. [ ] Accept email invitation
2. [ ] Complete profile setup
3. [ ] Download Linear desktop app (optional)
4. [ ] Download Linear mobile app (optional)
5. [ ] Review team workflows and labels
6. [ ] Set up keyboard shortcuts

## Integrations

Linear integrates with:
- GitHub (link PRs to issues)
- Slack (notifications)
- Figma (design links)

## Access Removal

1. Go to Settings → Members
2. Find member and remove from workspace
3. Reassign any owned issues
