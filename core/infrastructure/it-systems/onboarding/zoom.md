# Zoom Onboarding

## Overview

Zoom is used for teleconferencing, video meetings, and client calls.

- **Lead Account:** hello@arcanian.cloud
- **Auth Method:** SSO/Email

## Onboarding Steps

### 1. Create Zoom Account

Option A - Join existing workspace:
1. Admin sends invitation from hello@arcanian.cloud account
2. New member receives email invitation
3. Click link to join and create account

Option B - Link existing account:
1. New member has existing Zoom account
2. Admin adds them to the workspace

### 2. Account Configuration

New member should:

1. Complete profile (name, photo)
2. Set up Personal Meeting ID (PMI)
3. Configure audio/video settings
4. Download Zoom desktop app

### 3. License Assignment

| Role | License Type | Features |
|------|--------------|----------|
| Co-founder | Pro/Business | Host unlimited meetings |
| Co-worker | Pro/Business | Host unlimited meetings |
| Guest | Basic | 40-min limit on hosted meetings |

### 4. Integration Setup (Optional)

- Calendar integration (Google Calendar)
- Asana integration for meeting links

### 5. New Member Tasks

1. [ ] Accept Zoom workspace invitation
2. [ ] Complete profile setup
3. [ ] Download Zoom desktop app
4. [ ] Download Zoom mobile app
5. [ ] Test audio/video settings
6. [ ] Set up calendar integration

## Meeting Best Practices

- Use waiting room for client calls
- Enable meeting passwords
- Record important meetings (with consent)

## Access Removal

1. Log in to Zoom admin portal
2. Go to User Management → Users
3. Find user and remove from account
4. Optionally transfer scheduled meetings
