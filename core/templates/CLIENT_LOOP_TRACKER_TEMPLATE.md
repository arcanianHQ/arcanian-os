## Rules

- Every loop has a **cadence** (daily / weekly / monthly / quarterly / per-project).
- Every loop has a **last run date**. If a loop has been missed twice in a row, it has drifted — either re-install or retire it.
- Retiring a loop is fine. Drifting into silence is not. Name the choice.
- Loop cadence respects the pairing above. A weekly ritual that fights the client contact's natural cadence will drift.
