## Rules

- Three to seven Commitments. No more. If you have ten, you have none.
- Each Commitment has **one owner**. Two owners = no owner.
- Each Commitment has **one measurable outcome** — a line that tells you done or not done.
- At quarter end, each Commitment is **Done / Not Done**. No "mostly done." No "rolling forward" unless you explicitly restart it next quarter.
- If a Commitment's outcome changes mid-quarter, log the change in `EVENT_LOG.md` with the reason.
