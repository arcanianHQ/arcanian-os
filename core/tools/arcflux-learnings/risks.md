# Risk Register

Track project risks, their impact, and mitigation strategies.

## Impact Levels

| Level | Description |
|-------|-------------|
| 🔴 Critical | Project failure, major deadline miss |
| 🟠 High | Significant delay or quality impact |
| 🟡 Medium | Notable but manageable impact |
| 🟢 Low | Minor inconvenience |

## Status

| Status | Description |
|--------|-------------|
| ⚪ Open | Risk identified, not yet addressed |
| 🔵 Mitigated | Mitigation in place |
| ✅ Closed | Risk resolved or no longer relevant |
